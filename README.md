# countdown
Countdown Program

A beginner-friendly Python project that demonstrates how to use a while loop for countdown counting.

📌 Description

This program starts from the number 8 and keeps decreasing the value by 1 until it reaches 0.

It prints each number during the countdown.

 Concepts Used

while loop
Variables
Decrement Operator (-=)
Comparison Operators
print() function

💻 Code

num = 8

while num > -1:
    print(num)
    num -= 1

▶️ Example Output

8
7
6
5
4
3
2
1
0
